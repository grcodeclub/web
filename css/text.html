white-space

Η ιδιότητα white-space στην CSS ελέγχει πώς το κείμενο σε ένα στοιχείο διαχειρίζεται τα κενά, τα νέα γραμμές και την αναδίπλωση κειμένου. Ρυθμίζει τη συμπεριφορά του κειμένου μέσα σε ένα στοιχείο όσον αφορά το πώς αποτυπώνονται οι κενές γραμμές και οι αλλαγές γραμμής.

Χρησιμοποιείται για να επιτρέπει την κανονική αναδίπλωση κειμένου και τη διαχείριση κενών.

 white-space: normal;

Χρησιμοποιείται για να εμποδίσει την αναδίπλωση του κειμένου και να διατηρήσει το περιεχόμενο σε μία γραμμή.

 white-space: nowrap;

Χρησιμοποιείται για να εμφανίσει το κείμενο με την ίδια μορφή όπως στο HTML, συχνά για κώδικα ή preformatted text.

white-space: pre;

Χρησιμοποιείται για να διατηρήσει την αρχική μορφή του κειμένου ενώ επιτρέπει την αναδίπλωση για να αποφευχθεί η υπερβολική οριζόντια κύλιση.

white-space: pre-wrap;

Χρησιμοποιείται για να διατηρήσει τις αλλαγές γραμμής από το HTML, ενώ επιτρέπει την αναδίπλωση του κειμένου.

white-space: pre-line;

Χρησιμοποιείται όταν θέλεις το στοιχείο να ακολουθεί την ίδια ρύθμιση white-space με το γονικό του στοιχείο.

white-space: inherit;
