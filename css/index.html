CSS

  • Κείμενο και Ιδιότητες

Η ιδιότητα width καθορίζει το πλάτος ενός στοιχείου.

div { width: 200px;}

Η ιδιότητα height καθορίζει το ύψος ενός στοιχείου.

div {  height: 100px;}

Η ιδιότητα margin καθορίζει το κενό (margin) γύρω από ένα στοιχείο.

div {margin: 20px;}
div {margin-top: 20px;}
div {margin-bottom: 20px;}
div {margin-right: 20px;}
div {margin-left: 20px;}

Η ιδιότητα padding καθορίζει το εσωτερικό κενό (padding) ενός στοιχείου.

div {padding: 20px;}
div {padding-left: 20px;}
div {padding-right: 20px;}
div {padding-top: 20px;}
div {padding-bottom: 20px;}

Η ιδιότητα border καθορίζει το περίγραμμα (border) ενός στοιχείου.

div {border: 2px solid black;}

Η ιδιότητα background-color καθορίζει το χρώμα φόντου ενός στοιχείου 

div {
  background-color: lightgray;
}

Η ιδιότητα float επηρεάζει τον τρόπο τοποθέτησης ενός στοιχείου και τον τρόπο που τα γύρω στοιχεία το περικλείουν.

/* Το στοιχείο θα αριστερίζει και τα γύρω στοιχεία θα προσαρμόζονται στα δεξιά του. */
.elementp1{float :left;}

/* Το στοιχείο θα δεξιοτεντεί και τα γύρω στοιχεία θα προσαρμόζονται στα αριστερά του. */
.element2{float: right;}

/* Το στοιχείο θα παραμείνει σταθερά στην θέση του ανεξάρτητα από την κύλιση της σελίδας. 
Θα είναι πάντα ορατό ακόμα και κατά την κύλιση της σελίδας. */
.element3{float: fixed;}

/*  Το στοιχείο δεν θα αλλάξει τη θέση του σε σχέση με την φυσική ροή.*/
.element4{position: none;}

Η ιδιότητα opacity καθορίζει τη διαφάνεια του στοιχείου.

div {opacity: 0.7;}

Η ιδιότητα list-style χρησιμοποιείται για να καθορίσει την εμφάνιση των στοιχείων λίστας, όπως τα bullet points ή αριθμημένα σημεία.

/* τετράγωνα bullet points εσωτερικά */
ul {list-style: square inside;}

/* αριθμημένα σημεία εξωτερικά */
ol { list-style: decimal outside;}

/* κύκλοι bullet points */
ul.circle-list {list-style: circle;}

/* καμία εμφάνιση bullet points */
ul.none-list {list-style: none;}

Η ιδιότητα display ελέγχει τον τρόπο προβολής των στοιχείων στην ιστοσελίδα.

/*Το στοιχείο εμφανίζεται ως block-level στοιχείο (καταλαμβάνει όλο το πλάτος του διαθέσιμου χώρου).*/
.element1{display:block;}
/*Το στοιχείο εμφανίζεται ως inline στοιχείο (καταλαμβάνει μόνο το χώρο που απαιτεί το περιεχόμενό του).*/
.element2{display: inline;}
/*Συνδυασμός των χαρακτηριστικών των inline και block.*/
.element3{display: inline-block;}
/*Εφαρμόζει ευέλικτο layout με το Flexbox model.*/
.element3{display: flex;}
/*Εφαρμόζει ευέλικτο layout με το CSS Grid model.*/
.element4{display: grid;}
/*Το στοιχείο δεν εμφανίζεται*/
.element5{display: none;}

Η ιδιότητα position ελέγχει τον τρόπο τοποθέτησης των στοιχείων στη σελίδα. 

/* Η θέση του στοιχείου καθορίζεται από τη φυσική ροή του 
κειμένου και τοποθετείται κανονικά στη σελίδα. */
.elementp1{position :static;}

/* Το στοιχείο θα τοποθετηθεί σε σχέση με το πρώτο πάνω στοιχείο με non-static θέση. 
Αν δεν υπάρχει τέτοιο στοιχείο, θα τοποθετηθεί σε σχέση με τον παράγοντα θέσης του body. */
.element2{position :absolute;}

/* Το στοιχείο θα παραμείνει σταθερά στην θέση του ανεξάρτητα από την κύλιση της σελίδας. 
Θα είναι πάντα ορατό ακόμα και κατά την κύλιση της σελίδας. */
.element3{position :fixed;}

/* Το στοιχείο θα συμπεριφέρεται ως relative μέχρις ότου φτάσει
 σε μια καθορισμένη θέση, οπότε γίνεται fixed. Είναι ένας συνδυασμός των relative και fixed.*/
.element4{position :sticky;}

/* Το στοιχείο θα τοποθετηθεί σε σχέση με την αρχική του θέση, 
αλλά δεν θα επηρεάσει την θέση των άλλων στοιχείων. */
.element4{position :relative;}

/*Επαναφέρει την τιμή της ιδιότητας position στην αρχική της τιμή.*/
.element4{position :initial;}

/*Κληρονομεί την τιμή της ιδιότητας position από το γονικό στοιχείο.*/
.element4{position :inherit;}

/*Επαναφέρει την τιμή της ιδιότητας position στο default για το εκάστοτε στοιχείο.*/
.element4{position: unset;}

Class και ID: Χρησιμοποιούνται για την επιλογή συγκεκριμένων στοιχείων.

.class-name {
    property: value;
}
#id-name {
    property: value;
}

Επιλογή στοιχείων σε συγκεκριμένες καταστάσεις.

//* :hover: Όταν το ποντίκι πάει πάνω από ένα στοιχείο, μπορεί να αλλάξει το χρώμα του κειμένου. */
button:hover {
    background-color: lightgrey;
}

/* :active: Όταν ένας σύνδεσμος πατιέται, μπορεί να αλλάξει το χρώμα του προσωρινά. */
a:active {
    color: red;
}
                        
/* :focus: Όταν ένα πεδίο εισαγωγής λαμβάνει εστίαση, μπορεί να αλλάξει το περίγραμμα.  */
input:focus {
    border: 2px solid green;
}

/* :first-child: Το πρώτο παιδί ενός γονικού στοιχείου να έχει χρώμα purple. */
p:li:first-child {
    color: purple;
}  

/* :last-child: Το τελευταίο παιδί ενός γονικού στοιχείου μπορεί να έχει χρώμα orange. */
p:last-child {
    color: orange;
}

/* :nth-child(n): Εφαρμόζεται σε ένα στοιχείο που είναι το n-οστό παιδί του γονικού του. */
p:nth-child(odd) {
    background-color: lightgray;
}
