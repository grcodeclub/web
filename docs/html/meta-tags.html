HTML Meta Tags

Στην HTML (HyperText Markup Language), η ετικέτα <meta> χρησιμοποιείται για να παρέχει metadata ή πληροφορίες σχετικά με το έγγραφο HTML.
Τα metadata είναι πληροφορίες που δεν εμφανίζονται απευθείας στην ιστοσελίδα, αλλά παρέχουν στους browser πληροφορίες σχετικά με το περιεχόμενο της σελίδας. 

<meta charset='utf-8'>
<meta name="keywords" content="HTML, CSS, JavaScript, website">

Καθορίζει τον συγγραφέα του εγγράφου.

<meta  name="author" content="Το όνομά σας">

Δίνει οδηγίες στα ρομπότ των μηχανών αναζήτησης για το πώς να χειριστούν τη σελίδα.

<meta name="robots" content="index, follow">

Αυτόματη ανανέωση της σελίδας κάθε 30 δευτερόλεπτα.

<meta http-equiv="refresh" content="30">

Καθορίζει το χρώμα θέματος για προγράμματα περιήγησης που υποστηρίζουν την αλλαγή χρώματος.

<meta name="theme-color" content="#317EFB">

Επαληθεύει την ιδιοκτησία της σελίδας για την Google.

<meta name="google-site-verification" content="κωδικός-επαλήθευσης">

Καθορίζει την επαφή για την ιστοσελίδα.

<meta name="contact" content="info@example.com">

Διασφαλίζει ότι η σελίδα θα εμφανίζεται σωστά στον Internet Explorer.

<meta http-equiv="X-UA-Compatible" content="IE=edge">

Καθορίζει τη διαμόρφωση της προβολής για κινητές συσκευές.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Δεν μπορεί να γίνει από τον user ZoumIn ZouOut

<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">

Ανακατευθύνει αυτόματα σε άλλη URL μετά από 5 δευτερόλεπτα.

<meta http-equiv="refresh" content="5; url=https://example.com">

Η URL της σελίδας, όπως θα εμφανίζεται στα socia.

<meta property="og:url" content="https://grcodeclub.github.io">

Καθορίζει τον τίτλο που θα εμφανίζεται όταν η σελίδα μοιράζεται στα socia.

<meta property="og:title" content="GrCode Club">

Περιγραφή που θα εμφανίζεται στο Facebook όταν μοιράζεται η σελίδα στα social.

<meta property='og:description' content='Περιγραφή'>

Η URL της εικόνας που θα εμφανίζεται όταν η σελίδα μοιράζεται στα social.

<meta property="og:image" content="PATH">

Καθορίζει τον τύπο της σελίδας για τα Open Graph(og). Συνήθως χρησιμοποιείται η τιμή "website" για γενικές σελίδες ή "article" για άρθρα.

<meta property="og:type" content="website">

Καθορίζει τη γλώσσα και τη γεωγραφική περιοχή του περιεχομένου.

<meta  property="og:locale" content="el_GR"

Καθορίζει το όνομα του ιστότοπου.

<meta property="og:site_name" content="Η Ιστοσελίδα μου">
