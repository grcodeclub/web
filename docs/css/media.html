@media

Στην CSS, η @media είναι μια εντολή που σου επιτρέπει να εφαρμόζεις συγκεκριμένους κανόνες στυλ μόνο όταν πληρούνται ορισμένες συνθήκες, όπως το μέγεθος της οθόνης, η διάσταση του περιβάλλοντος ή η κατάσταση της συσκευής. Αυτό το σύστημα χρησιμοποιείται κυρίως για να δημιουργήσεις responsive designs, δηλαδή σχεδιασμούς που προσαρμόζονται στις διάφορες οθόνες και συσκευές.

Βασική Σύνταξη

@media media-type and (condition) {
/* CSS rules here */
}

Ορίζει ελάχιστο ή μέγιστο ύψος της οθόνης.

Χρησιμοποιείται για στυλ που σχετίζονται με το ύψος του παραθύρου προγράμματος περιήγησης.

@media screen and (max-width: 600px) {
body {
background-color: lightblue;
}
}

Ορίζει την κατεύθυνση της οθόνης (portrait ή landscape).

Χρησιμοποιείται για προσαρμογές της στυλ ανάλογα με την κατεύθυνση της οθόνης.

@media screen and (orientation: portrait) {
body {
background-color: lightyellow;
}
}

Ορίζει την ανάλυση της οθόνης, συνήθως με μονάδα μέτρησης σε DPI (dots per inch) ή DPCM (dots per centimeter).

Χρησιμοποιείται για να εφαρμόσει στυλ βασισμένα στην ανάλυση της οθόνης, όπως για να βελτιώσεις την εμφάνιση εικόνων σε οθόνες υψηλής ανάλυσης.

@media screen and (min-resolution: 192dpi) {
img {
border: 2px solid black;
}
}

Ορίζει τη σχέση διαστάσεων (πλάτος προς ύψος) της οθόνης ή του περιβάλλοντος.

Χρησιμοποιείται για να προσαρμόσει το στυλ ανάλογα με την αναλογία διαστάσεων της οθόνης ή του στοιχείου.

@media screen and (aspect-ratio: 16/9) {
body {
background-color: lightgray;
}
}

Ορίζει τον αριθμό των χρωμάτων που υποστηρίζονται από την οθόνη.

Χρησιμοποιείται για να προσδιορίσεις αν η οθόνη υποστηρίζει βασικά ή προηγμένα χρώματα.

@media screen and (color) {
p {
color: blue;
}
}

Ορίζει τον αριθμό των χρωμάτων που υποστηρίζονται από την οθόνη, αλλά με διαφορετική μέτρηση από την ιδιότητα color.

Χρησιμοποιείται για συσκευές με περιορισμένα χρώματα

@media screen and (color-index: 256) {
body {
background-color: lightpink;
}
}

Ορίζει το πλάτος ή το ύψος της οθόνης της συσκευής, ανεξάρτητα από το μέγεθος του παραθύρου του προγράμματος περιήγησης.

Χρησιμοποιείται για στυλ που πρέπει να εφαρμόζονται ανάλογα με τις διαστάσεις της οθόνης της συσκευής.

@media screen and (device-width: 768px) {
body {
background-color: lightblue;
}
}

Ορίζει την αναλογία διαστάσεων της συσκευής.

Χρησιμοποιείται για να εφαρμόσει το στυλ, ανάλογα με την αναλογία διαστάσεων της οθόνης της συσκευής.

@media screen and (device-aspect-ratio: 16/9) {
body {
background-color: lightcoral;
}
}

Εφαρμογή Στυλ για Εκτύπωση

@media print {
body {
font-size: 12pt;
}
}
