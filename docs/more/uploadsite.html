A Record

Η Α εγγραφή (A record) ή αλλιώς Address record καθορίζει ποια IP διεύθυνση αντιστοιχεί σε ένα όνομα χώρου.

CNAME

Η Cname εγγραφή (Canonical Name record) συνδέει το αληθινό κανονικό (canonical) όνομα με ψεύτικα (alias) ονόματα που χρησιμοποιεί ο ίδιος ο υπολογιστής.

Mail Exchanger Record (MX Record)

H MX εγγραφή (Mail Exchanger Record) αφορά μόνο την mail υπηρεσία μας και μας επιτρέπει να ανακατευθύνουμε τα emails που μας στέλνουν σε λογαριασμούς του domain μας σε οποιοδήποτε mail server θέλουμε εμείς.

Πιστοποιητικά SSL
Γιατί SSL;

Το πιστοποιητικό SSL δημιουργεί μια ασφαλή σύνδεση μεταξύ του server που φιλοξενεί ένα website και του browser του επισκέπτη, προστατεύοντας έτσι όλες τις πληροφορίες που μεταφέρονται από και προς το website, όπως username, κωδικοί, αριθμοί πιστωτικών καρτών κτλ.
