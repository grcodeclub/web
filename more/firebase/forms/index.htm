Firebase - Forms

1. Δημιούργησε ένα Firebase project

  1. Άνοιξε τον περιηγητή σου και πήγαινε στη Firebase Console.
  2. Κάνε κλικ στο κουμπί "Add project" (Προσθήκη project).
  3. Εισήγαγε ένα όνομα για το project σου και κάνε κλικ στο "Continue" (Συνέχεια).
  4. Ρυθμίσεις Google Analytics (Προαιρετικά)
  5. Όταν ολοκληρωθεί, κάνε κλικ στο "Continue" (Συνέχεια) για να μεταφερθείς στον πίνακα ελέγχου του νέου project σου.
  6. Στον πίνακα ελέγχου του project, μπορείς να προσθέσεις μια εφαρμογή (web, iOS, Android) κάνοντας κλικ στο εικονίδιο της αντίστοιχης πλατφόρμας και ακολουθώντας τις οδηγίες.
  7. Μετά την προσθήκη της εφαρμογής, θα δεις έναν κώδικα με τις παραμέτρους ρύθμισης (apiKey, authDomain, projectId, κ.λπ.). Αντέγραψε αυτόν τον κώδικα και χρησιμοποίησέ τον στην εφαρμογή σου.

3. Παράμετροι Ρύθμισης Firebase

Είναι το κλειδί API της εφαρμογής σας. Χρησιμοποιείται για την ταυτοποίηση της εφαρμογής με το Firebase.

apiKey: "YOUR_API_KEY"

Είναι το domain που χρησιμοποιείται για την πιστοποίηση των χρηστών. Συνήθως έχει τη μορφή your-app-id.firebaseapp.com.

authDomain: "YOUR_AUTH_DOMAIN"

Είναι το μοναδικό αναγνωριστικό του project σας στο Firebase. Κάθε project στο Firebase έχει ένα μοναδικό αναγνωριστικό.

projectId: "YOUR_PROJECT_ID"

Είναι το χώρο αποθήκευσης που χρησιμοποιείται από το Firebase Storage. Συνήθως έχει τη μορφή your-app-id.appspot.com.

storageBucket: "YOUR_STORAGE_BUCKET"

Είναι το αναγνωριστικό αποστολέα που χρησιμοποιείται για την αποστολή μηνυμάτων μέσω του Firebase Cloud Messaging.

messagingSenderId: "YOUR_MESSAGING_SENDER_ID"

Είναι το αναγνωριστικό της εφαρμογής σας. Χρησιμοποιείται για να διακρίνει την εφαρμογή σας από άλλες εφαρμογές.

appId: "YOUR_APP_ID"

3. Δημιούργησε το Script

  1. firebaseConfig: Περιέχει τις ρυθμίσεις που απαιτούνται για τη σύνδεση με το project σας στο Firebase. Αυτές οι ρυθμίσεις περιλαμβάνουν το apiKey, το authDomain, το projectId, το storageBucket, το messagingSenderId και το appId.
  2. firebase.initializeApp(firebaseConfig): Αρχικοποιεί το Firebase με αυτές τις ρυθμίσεις.
